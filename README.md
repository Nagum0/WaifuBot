# Commands
- w!get <...>
    - Sends in a random image from google images by the inputted search term;
- w!bible
    - Sends in a random bible verse;
- w!rps <...>
    - Plays a round of rock paper scissors with you;
- w!wcoins
    - Returns the amount of **Wcoins** you own on the server;
